2. Variáveis:

O que são variáveis? São espaços nomeados na memoria que podem guardar valores, seja de operações ou resultados.

No python, para se criar uma variável, basta digitar um nome e receber algum valor

Ex.:

                    

          nome = "Pedro"

Na hora de escolher o nome para a variável você deve-se prestar atenção em algumas regras da linguagem:

  • O nome da variável não pode começar com número
  • O nome da variável não pode conter espaços
  • O nome da variável não pode ser palavras reservadas da linguagem
  • O nome da variável não pode ter acentos ou caracteres especiais

Forma errada:

                    

        ❌ 1nome = "carlos"

        ❌ número primo = 3

        ❌ ação = 'jogar'

Forma certa:

                    

        ✅ nome1 = "Carlos"

        ✅ numero_primo = 3

        ✅ numerosPrimos = 4

        ✅ acao = 'jogar'

02 - Tipos e simbolos

1 - Tipos primitivos:

No python existem os seguintes tipos:

  • int() - Ele é usado apénas para números inteiros e converter bases

Ex.:

                        

        ✅ 3 é int()

        ✅ -4 é int()

        ❌ 3.5 não é int
  • float() - Ele é usado quando temos números flutuantes (com virgula) e converte bases

Ex.:

                        

        ✅ 3.0 é float()

        ✅ -1.0 é float()
  • str() - Ele é o tipo que representa tudo que está dentro das aspas " ", ' ' e ele também pode ser usado para converter bases int e float para str

Ex.:

                    

    ✅ "Carlos algustos" é str

    ✅ '235' é str 

    ❌ carlos algusto  não é str 
 por estár fora das aspas

    ❌ 235 não é str, é int
  • Boolean - Nessa classe só existe dois, True e False. Eles são operadores logicos e usados para criar e verificar logicas
