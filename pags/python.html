2. Variáveis:

O que são variáveis? São espaços nomeados na memoria que podem guardar valores, seja de operações ou resultados.

No python, para se criar uma variável, basta digitar um nome e receber algum valor

Ex.:

                    

          nome = "Pedro"

Na hora de escolher o nome para a variável você deve-se prestar atenção em algumas regras da linguagem:

  • O nome da variável não pode começar com número
  • O nome da variável não pode conter espaços
  • O nome da variável não pode ser palavras reservadas da linguagem
  • O nome da variável não pode ter acentos ou caracteres especiais

Forma errada:

                    

        ❌ 1nome = "carlos"

        ❌ número primo = 3

        ❌ ação = 'jogar'

Forma certa:

                    

        ✅ nome1 = "Carlos"

        ✅ numero_primo = 3

        ✅ numerosPrimos = 4

        ✅ acao = 'jogar'

02 - Tipos e simbolos

1 - Tipos primitivos:

No python existem os seguintes tipos:

  • int() - Ele é usado apénas para números inteiros e converter bases

Ex.:

                        

        ✅ 3 é int()

        ✅ -4 é int()

        ❌ 3.5 não é int
  • float() - Ele é usado quando temos números flutuantes (com virgula) e converte bases

Ex.:

                        

        ✅ 3.0 é float()

        ✅ -1.0 é float()
  • str() - Ele é o tipo que representa tudo que está dentro das aspas " ", ' ' e ele também pode ser usado para converter bases int e float para str

Ex.:

                    

    ✅ "Carlos algustos" é str

    ✅ '235' é str 

    ❌ carlos algusto  não é str 
 por estár fora das aspas

    ❌ 235 não é str, é int
  • Boolean - Nessa classe só existe dois, True e False. Eles são operadores logicos e usados para criar e verificar logicas

2 - Operadores Matematicos:

No python existe alguns operadores matematicos basicos, são eles:

  • + (adição) - Ele é representado pelo simbolo "+", pode ser usado para unir strs e números: 2 + 2 = 4 | "2" + "2" = "22"
  • - (subtração) - Ele é representado pelo simbolo "-" : 2 - 2 = 0
  • * (multiplicação) - Ele é representado pelo simbolo de "*" : 2 * 6 = 12 | print("==="*10)
  • ** (exponenciação) - Ele é representado pelo simbolo de "**" : 3 ** 3 = 27
  • / (divisão) - Ele é representado pelo simbolo de "/" : 12 / 6 = 2
  • // (inteiro da divisão) - Ele é representado pelo simbolo de "//" e é o valor todal da divisão : 5 / 2 = 2
  • % (resto da divisão) - Ele é representado pelo simblo de "%" e é o valor que sobra de uma divisão : 5 / 2 = 1 | 4 / 2 = 0

3 - Operadores Lógicos

Os operadores lógicos são a base inicial da programação e da lógica de criação de sistemas mais complexos.

No Python nois temos os operadores:

1- and (e) - Ele é usado para comparar se dois elementos cumprem os requisitos. Se os dois cumprirem, ele ira retornar "True". Se um deles não cumprir os requisitos, ele ira retornar "False"

Ex.:

                    

  valor1 = 0

  valor2 = 4

  # Use essas variaveis como base

  ❌ valor1 = 4 and valor2 = 4 ?

  # É False pelo fato do valor1
    não ser igual a 4 e os dois
    tem que ser verdadeiro

  ✅ valor1 = 0 and valor2 = 4 ?

  # É True pelo fato dos dois
  valores sã0 verdadeiros

2 - or (ou) - Ele é usado para comparar dois elementos, se os dois elementos cumprirem os requisitos, ou um cumprir os requisitos, ele retorna "True". Se os dois valores não cumprirem os requisitos ele retorna "False"

Ex.:

                    

  valor1 = 0

  valor2 = 4

  # Use essas variaveis como base

  ❌ valor1 = 40 or valor2 = -4 ?

  # É False pelo fato dos dois 
    valores serem falsos

  ✅ valor1 = 0 or valor2 = 4 ?

  # É True pelo fato dos dois
    valores serem verdadeiros

  ✅ valor1 = 48 and valor2 = 4 ?

  # É True pelo fato de um dos
  valores ser verdadeiro

3 - == (igualdade) - No Python o sinal de "=" significa "recebe" e o sinal "==" significa "igaul?". Caso o valor for igaul ao requisito, ele retorna True, caso contrario le retorna False 

                    

  valor1 = 0

  valor2 = 4

  # Use essas variaveis como base

  ❌ valor1 == valor 2 ?

  ✅ valor 1 == 0 ?

4 - != (diferente) - No Python temos um simbolo que representa a diferença "!=". Ele serve para comparar se um valor é diferente de outro, se for ele retorna "True", se não ele retorna "False"

Ex.:

                        
  valor1 = 0

  valor2 = 4

  # Use essas variaveis como base
  
  ✅ valor1 != valor 2 ?

  ❌ valor 1 != 0 ?
