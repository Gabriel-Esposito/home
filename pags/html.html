Tags

Elemento HTML é uma parte de uma página da internet. Um elemento é qualquer objeto identificável dentro de um documento, por exemplo, um caractere, palavra, imagem, parágrafo ou célula de planilha.

As tags em html devem ser abertas e fechadas pelos sinais de maior (>) e menor (<)

Ex:

                    
                    ❌ <p><p>
                    ❌ <p <p>
                    ✅ <p></p>

Aqui estão algumas tags do HTML5:

  • h1 / h6 : usado para criar uma hierarquia de título e vai do h1 até o h6
  • p : usado para criar um parágrafo na pag
  • img : usado para adicionar uma imagem na pag
  • video : usada para adicionar um vídeo na pag
  • strong : usado para deixar uma parte do texto em negrito
  • b : usado para deixar uma parte do texto em negrito, porém, não é semântico
  • em : usado para deixar uma parte do texto em itálico
  • i : usado para deixar uma parte do texto em itálico, porém, não é semântico
  • mark : usado para criar uma marcação em um trecho do texto
  • small : usada para deixar um trecho do texto pequeno
  • del : vai criar uma linha sobre o um trecho do texto
  • ins : usado para colocar uma linha a baixo do texto
  • U : usado para colocar uma linha a baixo do texto, porém, não semântico
  • sup : usado para elevar algo
  • sub : usado para abaixar algo
  • q : usado para marcar uma citação de alguel no texto
  • blockquote : usado para marcar uma citação de alguem no texto, porém, podemos colocar um link para a citação
  • abbr : usada para criar uma abreviação
  • bdo : usada para inverter um trecho no texto
  • ol : usado para criar uma lista numérica
  • ul : usado para criar uma lista com simbolos
  • li : usado para adicionar um texto dentro de uma lista
  • dl : usado para criar uma lista de definições
  • dd : udado para dar uma descrição para os elementos da lista de definições
  • a : usado para colocar links, criar botões, baixar arquivos, etc... no cite
  • picture : usado para criar imagens dinâmicas
  • source : usado para criar uma dinamica em algum elemento qunado atinjir um tamanho
  • audio : cria junto com a tag source um arquivo de áudio no site
  • pre : não modifica o estilo do texto dentro da tag
  • code : usado para escrever códigos sem serem modificados
  • iframe : usado para importar vídeos e abrir sites dentro de seu proprio site
  • span : usada para separar umna parte de um site e modificá-lo com scripts ou css

Existem algumas tag que não precisão serem fechadas:

  • br : usada para quebrar uma linha em um texto
  • hr : usado para criar uma linha da horizontal do site
  • link : usado para colocar favicons no site, importar css, etc...
  • script : usado para importar e usar scripts

Divs

As divs são uma parte muito importante de um site, afinal, são elas que são responsáveis por dividir um site em blocos que podem ser manipulados pelas CSS3, JavaScript e etc...

Existem vários tipos de divs semanticas que podem ser usadas para melhor compreensão de seu código

Divs semanticas

  • header : Ele é usado como o cabeçalho de seu site
  • nav : Usado para colocar seus links de navegação no site
  • main : Usado como bloco central do site
  • section : Usada para criar seções no site
  • article : Usada para colocar artigos relacionados aos assustos do site
  • footer : Usado como rodapé do site, onde ficara os créditos finais

Um exemplo de um site semântico

Ex:

                
                <header>
                    <nav>
                    <a href="link"> Home </a>
                    </nav>
                </header>

                <main>
                    <section>
                        <article>
                            (conteúdo)
                        </article>
                    </section>
                </main>

                <footer>
                    (créditos)
                </footer>

Uma div é um bloco ou pedaço do site que pode ser mudado, click no quadrado amarelo!

Ex:

Podemos colocar divs dentro de outras divs, assim criando uma hierarquia de divs

Podemos mover as divs usando o CSS3

Como usar:

Basta colocar a tag div e criar algo dentro dela

Ex:

                
                <div>
                    <p> Olá Mundo!</p>
                </div>

OBS: para notar a diferença será necessário aplicar um código CSS.

Hierarquia de divs:

basta colocar uma div dentro da outra e as dar um id ou uma class.

                
                <div id="pai">
                    <div=""filho>
                        <p> Olá Mundo!</p>
                    </div>
                </div>

OBS: IDs e ClASSs serão explicados depois.

Class / id

CLASS {.}:

As vezes precisamos mudar vários elementos de um site, porém, eles terão a mesma config, como fazer isso sem o código ficar grande e pesado?

Basta usar a Class!
Podemos dar uma class para os elementos de mesma config, assim os modificando.

Class São usadas para varias modificações e também uma parte importante para juntar o Front-end e o Back-end de teu site!

Como usar class

Escolha os elementos que preferir modificar colocando o seletor class e atribuindo um nome ao mesmo.

Ex:

                
            <p class="o_nome_de_sua_class">
                Olá mundo!
            </p>

Ex2:

                
        ❌  <p class="box 2"> Olá mundo </p>
        ❌  <p class='Box'> Olá mundo </p>
        ❌  <p clas="box"> Olá mundo </p>
        ✅  <p class="box_2"> Olá mundo </p>

A class deve ser escrito corretamente para não haver erro.
Evite dar espaços no nome de suas class!
Troque os espaços por (-) ou (_) letras maiúsculas também ajudão (BoxNova).

A class funciona melhor com vários elementos, porém, podemos atribuir a apenás um elemento

Ex:

                
    ✅  <p class="box_2"> parágrafo </p>
    ✅  <h4 class="Titulos"> titulo </h4>
    ✅  <a href="" class="box_2"> âncora </a>

ID {#}:

IDs, diferentes das class, o id normalmente é atríbuido a um elemento unico no site
Usados quando apenas um elemento específico vai ser mudado

Ex:

                
        ✅  <p id="maior"> Olá mundo </p>
        ❌  <p id="maior"> Olá mundo </p>
        ❌  <h1 id='maior> Olá mundo </h1>

Podemos juntar IDs e CLASSs para deixar o site mais dinamico.

Ex:

                
        ✅ <h1 class="Titulos">
            (conteúdo)
        </h1>
        <p class="Titulos" id="caixa-de-texto">
            (conteúdo)
        </p>
        <h4 class="Sub-Titulo">
            (conteúdo)
        </h4>
